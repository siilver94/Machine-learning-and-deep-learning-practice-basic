## CSS 선택자(Selector) 란?

선택자란 말 그대로 선택을 해 주는 요소입니다. 이를 통해 특정 요소들을 선택하여 스타일을 적용할 수 있게 됩니다.

<br/>

### 태그 선택자

`u`

`div`

`li`

<br/>

### 아이디 선택자

`#<아이디 이름>`

<br/>

### 클래스 선택자

`.<클래스 이름>.<클래스 이름>.<클래스 이름>`

<br/>

### 후손 선택자

`#meigen li`

<br/>

### 자식 선택자

`ul.itmes > li`
